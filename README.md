# 💊 Medicine Intake Monitoring System

## 📌 Overview
The **Medicine Intake Monitoring System** is a smart healthcare application that helps monitor whether patients take their medicines on time. It uses computer vision to detect user presence and sends alerts if medication is missed.

---

## 🚀 Features
- 👤 Face detection using MediaPipe
- 📷 Real-time monitoring via webcam
- ⏰ Time-based reminders (morning, afternoon, night)
- 📱 SMS/Call alerts using Twilio
- 🌐 Web interface using Flask
- 📊 Patient tracking and scheduling

---

## 🛠️ Tech Stack
- **Backend:** Python, Flask  
- **Computer Vision:** OpenCV, MediaPipe  
- **Alerts:** Twilio API  
- **Database:** SQLite / MySQL  
- **Scheduler:** APScheduler
